# dotfiles

## Requirements

* [git](https://git-scm.com)
* [stow](https://www.gnu.org/software/stow/)

## Installation 

Clone this repo to `${HOME}/.dotfiles` and run the following.

```
cd ${HOME}/.dotfiles
./internal/setup.sh
```
