# dotfiles

## Requirements

* [git](https://git-scm.com)
* [rcm](https://github.com/thoughtbot/rcm)

## Installation 

Clone this repo to `${HOME}/.dotfiles` and run the following.

```
${HOME}/.dotfiles/internal/setup.sh
```
